# MXLRCSimulator
简易实现的歌词展示效果，随着音乐的进度，歌词也随着滚动，仅作学习之用。


# 主要实现

- 歌词随音乐的播放进度往上平滑滚动，并非跳动滚动
- 歌词随音乐同步显示，在唱的那句歌词，已唱部分文字高亮显示
- 拖动音乐进度条歌词同步平滑滚动
- 支持上下拖动歌词，拖动结束，音乐也同步播放到对应位置
- 上下拖动歌词过程中显示中间歌词行所对应的时间线
- 歌词总体显示效果类似于天天动听
